Módulo 5 — Eventos Independentes e Tentativas Repetidas

Quando um evento não altera a chance de outro — e como isso vira a distribuição binomial.

Teoria

📌 \(A\) e \(B\) são independentes se a ocorrência de um não influencia a probabilidade do outro. Formalmente, \(B\) é independente de \(A\) se \(P(B) = P(B\mid A)\), o que é equivalente a:

\[P(A \cap B) = P(A)\cdot P(B)\]

📌 Tentativas independentes repetidas. Se \(S\) é um espaço de probabilidade finito, o espaço de \(n\) tentativas independentes é \(S^n\) (n-uplas ordenadas de elementos de \(S\)), onde a probabilidade de cada n-upla é o produto das probabilidades de seus componentes.

Complemento (não está nas aulas, mas conecta diretamente com os exercícios da lista): quando um experimento binário (sucesso com probabilidade \(p\), fracasso com \(1-p\)) é repetido \(n\) vezes de forma independente, a probabilidade de exatamente \(k\) sucessos é dada pela distribuição binomial:

\[P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}\]
💡 Explicando bem devagar

Dois eventos são independentes quando um não "avisa" nada sobre o outro. Jogar uma moeda duas vezes: o resultado da 1ª jogada (cara ou coroa) não muda em nada a chance da 2ª jogada — elas são independentes. Já "está chovendo" e "tem gente com guarda-chuva na rua" NÃO são independentes: saber uma coisa muda a chance da outra.

Para eventos independentes, existe um atalho enorme: a chance dos DOIS acontecerem juntos é só multiplicar as chances de cada um: \(P(A\cap B) = P(A)\times P(B)\). Nada de somar ou subtrair, só multiplicar.

Quando você repete o MESMO experimento várias vezes (tipo jogar uma moeda 5 vezes, ou um jogador tentando acertar uma cesta 5 vezes), e cada tentativa é independente das outras, existe uma fórmula pronta chamada distribuição binomial para responder "qual a chance de dar certo exatamente \(k\) vezes em \(n\) tentativas". Ela tem 3 pedaços, cada um respondendo uma pergunta:

  • \(\binom{n}{k}\) = "de quantos JEITOS diferentes posso escolher QUAIS tentativas deram certo?"
  • \(p^k\) = "chance de acertar, multiplicada \(k\) vezes (uma vez pra cada acerto)"
  • \((1-p)^{n-k}\) = "chance de errar, multiplicada pelas vezes restantes que erraram"

Multiplique os três pedaços e pronto. É exatamente essa fórmula que vai aparecer o tempo todo daqui pra frente — e nos exercícios abaixo, cada acerto/erro possível é escrito por extenso (tipo "aaeee") pra você ver de onde vem o \(\binom{n}{k}\).

Exemplo 6 (aula)

a) 3 lançamentos de moeda. A={1º lance é cara}, B={2º lance é cara}, C={exatamente duas caras seguidas}. Quais pares são independentes? b) Tiro ao alvo: atirador A acerta com \(P=1/4\), atirador B acerta com \(P=2/5\). Ambos atiram. Qual a probabilidade de que pelo menos um acerte?
💡 Passo a passo bem simples

a) 3 moedas jogadas. A pergunta é: A (a 1ª deu cara), B (a 2ª deu cara) e C (deu exatamente 2 caras SEGUIDAS) — quais pares "não se afetam"? Para cada par, você calcula \(P(X\cap Y)\) (a chance real dos dois juntos, contando os casos) e compara com \(P(X)\times P(Y)\) (o que "deveria" dar se fossem independentes). Se bater, são independentes; se não bater, um afeta o outro.

Resultado curioso: A e B são independentes (faz sentido, moedas diferentes). A e C também. Mas B e C NÃO são — porque toda vez que C acontece (duas caras seguidas), a 2ª moeda OBRIGATORIAMENTE deu cara, então saber que C aconteceu já "avisa" que B também aconteceu. Isso é o oposto de independência.

b) Dois atiradores, cada um atira uma vez. "Pelo menos um acerta" de novo usa o truque do contrário: calcule a chance de os DOIS errarem (multiplicando as chances de erro de cada um, porque são independentes), e depois faça \(1-\)isso.

I. a) Resolução \(S=\{CCC,CCK,CKC,CKK,KCC,KCK,KKC,KKK\}\) (C=cara, K=coroa), 8 elementos equiprováveis.
\(A\) (1º é cara) = \(\{CCC,CCK,CKC,CKK\}\), \(P(A)=1/2\).
\(B\) (2º é cara) = \(\{CCC,CCK,KCC,KCK\}\), \(P(B)=1/2\).
\(C\) (exatamente duas caras seguidas, não três) = \(\{CCK, KCC\}\), \(P(C)=2/8=1/4\).
Testando cada par com \(P(X\cap Y)=P(X)P(Y)\):
\(A\cap B=\{CCC,CCK\}\), \(P=2/8=1/4\); \(P(A)P(B)=1/4\). Iguais → independentes.
\(A\cap C=\{CCK\}\), \(P=1/8\); \(P(A)P(C)=(1/2)(1/4)=1/8\). Iguais → independentes.
\(B\cap C\): ambos elementos de \(C\) (CCK e KCC) têm 2º lance = cara, logo \(B\cap C = C = \{CCK,KCC\}\), \(P=1/4\); \(P(B)P(C)=(1/2)(1/4)=1/8\). Diferentes → dependentes.
II. b) Resolução "Pelo menos um acerta" → complemento de "nenhum acerta".
Atiradores independentes → \(P(\text{nenhum}) = \) produto dos complementos.
\[P(\text{nenhum acerta}) = (1-\tfrac14)(1-\tfrac25) = \tfrac34 \cdot \tfrac35 = \tfrac{9}{20}\] \[P(\text{pelo menos um}) = 1 - \frac{9}{20} = \frac{11}{20}\]
E se a condição fosse...Mudança no raciocínio
b) Três atiradores em vez de dois, com \(P=1/4, 2/5, 1/3\)O complemento ganha um terceiro fator: \(P(\text{nenhum acerta}) = (3/4)(3/5)(2/3)\); "pelo menos um" continua sendo \(1\) menos esse produto.
Pergunta fosse "exatamente um acerta" (item b)Não basta o complemento: é preciso somar os 2 casos em que só A acerta ou só B acerta, cada um com o outro errando: \(P(A)(1-P(B)) + (1-P(A))P(B)\).
a) C fosse "exatamente uma cara em todo o experimento" em vez de "duas seguidas"C mudaria para \(\{CKK,KCK,KKC\}\), alterando P(C) e exigindo refazer os testes de independência com A e B.
III. Generalização 🔁 "Pelo menos um evento independente ocorre, dentre vários" quase sempre se resolve pelo complemento: \(1 - \prod_i (1-p_i)\).
No quadro a) P(A)=1/2, P(B)=1/2, P(C)=1/4
P(A∩B)=1/4=P(A)·P(B) → SÃO INDEPENDENTES
P(A∩C)=1/8=P(A)·P(C) → SÃO INDEPENDENTES
P(B∩C)=1/4 ≠ P(B)·P(C)=1/8 → NÃO SÃO INDEPENDENTES
b) P(a)=1/4, P(b)=2/5
P(ā)=3/4, P(b̄)=3/5
P(nenhum) = 3/4 · 3/5 = 9/20 = 0,45
P(pelo menos 1) = 1 - 9/20 = 11/20 = 0,55

Exemplo 7 (aula) — Corrida de cavalos, 2 competições

Três cavalos a, b, c com \(P(a)=\frac12, P(b)=\frac13, P(c)=\frac16\). Competem 2 vezes hoje. Calcule a probabilidade de cada elemento do espaço amostral.
💡 Passo a passo bem simples

3 cavalos correm 2 vezes no mesmo dia. Cada corrida é independente da outra (o resultado da 1ª não afeta a 2ª). Para saber a chance de, por exemplo, "cavalo a vence a 1ª corrida E cavalo b vence a 2ª", é só multiplicar a chance de cada coisa separada: \(P(a)\times P(b)\).

Existem \(3\times3=9\) combinações possíveis de resultado (quem ganha a 1ª, quem ganha a 2ª), e para cada uma delas você multiplica as duas chances envolvidas. No final, um jeito de conferir se fez tudo certo: somando as 9 probabilidades, o resultado TEM que dar exatamente 1 (100%) — porque com certeza algum desses 9 resultados vai acontecer.

I. Resolução Espaço de 2 tentativas: \(S^2 = \{a,b,c\}\times\{a,b,c\}\), 9 elementos.
Cada probabilidade = produto das probabilidades dos componentes:
\[P(a,a)=\tfrac12\cdot\tfrac12=\tfrac14 \quad P(a,b)=\tfrac12\cdot\tfrac13=\tfrac16 \quad P(a,c)=\tfrac12\cdot\tfrac16=\tfrac{1}{12}\] \[P(b,a)=\tfrac16 \quad P(b,b)=\tfrac13\cdot\tfrac13=\tfrac19 \quad P(b,c)=\tfrac13\cdot\tfrac16=\tfrac{1}{18}\] \[P(c,a)=\tfrac{1}{12} \quad P(c,b)=\tfrac{1}{18} \quad P(c,c)=\tfrac16\cdot\tfrac16=\tfrac{1}{36}\] Checagem: soma das 9 probabilidades = 1 → \(\frac14+\frac16+\frac{1}{12}+\frac16+\frac19+\frac{1}{18}+\frac{1}{12}+\frac{1}{18}+\frac{1}{36}=1\)
(denominador comum 36: \(9+6+3+6+4+2+3+2+1=36\)) ✓
E se a condição fosse...Mudança no raciocínio
3 competições em vez de 2O espaço passa a ser \(S^3\), com \(27\) elementos; cada probabilidade é o produto de 3 fatores em vez de 2 (ex.: \(P(a,a,a)=(1/2)^3=1/8\)).
Perguntar \(P(\text{mesmo cavalo vence as 2 corridas})\)Soma das probabilidades da diagonal: \(P(a,a)+P(b,b)+P(c,c) = 1/4+1/9+1/36 = 7/18\), em vez de listar todos os 9 pares.
II. Generalização 🔁 O espaço de \(n\) tentativas independentes de um mesmo experimento é sempre o produto cartesiano \(S^n\), e a probabilidade de cada n-upla é o produto das probabilidades individuais de cada componente — por isso a soma de todas as \(|S|^n\) probabilidades deve sempre fechar em 1, o que serve como checagem rápida do trabalho.
No quadro P(a)=1/2, P(b)=1/3, P(c)=1/6, S² = 9 elementos
P(a,a)=1/2·1/2=1/4=0,25
P(a,b)=P(b,a)=1/2·1/3=1/6=0,166...
P(a,c)=P(c,a)=1/2·1/6=1/12=0,0833...
P(b,b)=1/3·1/3=1/9=0,111...
P(b,c)=P(c,b)=1/3·1/6=1/18=0,0555...
P(c,c)=1/6·1/6=1/36=0,0277...
soma = 36/36 = 1 ✓

Exercícios da Lista III

Questão 17 — Cavalos a, b, c correm 3 vezes

\(P(a)=0.3, P(b)=0.5, P(c)=0.2\). a) Mesmo cavalo ganha as 3 corridas. b) Cada um dos 3 cavalos vence exatamente uma vez.
💡 Passo a passo bem simples

a) "O mesmo cavalo ganha as 3 corridas": pode ser sempre o a, ou sempre o b, ou sempre o c — três jeitos diferentes de acontecer, que nunca se misturam (excludentes). Calcule a chance de cada um (multiplicando a probabilidade daquele cavalo 3 vezes, já que são 3 corridas independentes) e SOME os três resultados.

b) "Cada cavalo vence exatamente uma vez em 3 corridas": pense em 3 corridas como 3 posições numa fila (1ª, 2ª, 3ª), e você precisa colocar a, b, c nessas posições sem repetir nenhum. Quantos jeitos de arrumar 3 coisas diferentes numa fila? \(3!=6\). Toda arrumação tem a MESMA chance, porque multiplicar \(P(a)\times P(b)\times P(c)\) dá o mesmo número não importa a ordem. Então: multiplique as 3 chances uma vez, e depois multiplique por 6 (as 6 ordens possíveis).

I–II. Dados e classificação Espaço de 3 tentativas independentes repetidas, \(S^3\) — mesma estrutura do Exemplo 7, mas com 3 competições em vez de 2, reconhecer isso primeiro evita recomeçar o raciocínio do zero.
III–IV. Resolução a) "Mesmo cavalo ganha as 3" → união de 3 eventos excludentes (a vence 3x, ou b, ou c).
Tentativas independentes → cada termo é o cubo da probabilidade individual: \[P = (0.3)^3 + (0.5)^3 + (0.2)^3 = 0.027 + 0.125 + 0.008 = 0.16\] b) "Cada cavalo vence exatamente uma vez" → permutação das 3 corridas entre os 3 cavalos.
Ordens possíveis: \(3! = 6\).
Cada ordem tem probabilidade \(0.3\times0.5\times0.2\) (produto comutativo, mesma para todas): \[P = 3! \times (0.3)(0.5)(0.2) = 6 \times 0.03 = 0.18\]
E se a condição fosse...Mudança no raciocínio
4 corridas em vez de 3 (item a)Cada termo vira a 4ª potência: \((0.3)^4+(0.5)^4+(0.2)^4 = 0.0081+0.0625+0.0016 = 0.0722\); com mais corridas, ficar preso ao mesmo cavalo fica mais raro.
"Exatamente 2 cavalos diferentes vencem (não os 3)" em vez do item bDeixa de ser uma simples permutação: é preciso escolher qual cavalo vence 2 vezes e qual vence 1, e contar as \(\binom{3}{2}\) ordens de intercalação para cada escolha — bem mais casos que os \(3!\) do item b.
Item b pedisse "b vence pelo menos 2 das 3 corridas"Passa a ser uma soma binomial: \(P(X\geq2)=\binom{3}{2}(0.5)^2(0.5)+\binom{3}{3}(0.5)^3\), não mais uma permutação simples.
V. Verificação Gabarito: a) 0.16, b) 0.18. Bate com o gabarito.
VI. Generalização 🔁 "O mesmo resultado se repete \(n\) vezes" → soma dos cubos (ou potências \(n\)) das probabilidades individuais. "Cada resultado distinto ocorre exatamente uma vez em \(n\) tentativas" → \(n!\) vezes o produto das probabilidades (regra da permutação combinada com tentativas independentes).
No quadro P(a)=0,3 P(b)=0,5 P(c)=0,2
a) P = (0,3)³+(0,5)³+(0,2)³ = 0,027+0,125+0,008 = 0,16
b) ordens (1ª,2ª,3ª): abc, acb, bac, bca, cab, cba → 3!=6
cada ordem: 0,3·0,5·0,2 = 0,03
P = 6·0,03 = 0,18

Questão 18 — Antônio, lance de 3 pontos, p=0.4, 5 tentativas

a) Exatamente duas vezes. b) Pelo menos uma vez.
💡 Passo a passo bem simples

Antônio tenta 5 vezes, cada tentativa acerta com chance 0,4 (40%) e erra com chance 0,6 (60%).

a) "Exatamente 2 acertos em 5 tentativas": pense em cada tentativa como uma letra, "a" (acertou) ou "e" (errou). Você precisa de sequências de 5 letras com exatamente 2 "a"s, tipo "aaeee", "aeaee", "eeeaa"... Existem \(\binom{5}{2}=10\) sequências assim diferentes (é só escolher QUAIS 2 das 5 tentativas foram acerto). Cada uma dessas 10 sequências tem a mesma chance: 0,4 multiplicado 2 vezes (pelos 2 acertos) vezes 0,6 multiplicado 3 vezes (pelos 3 erros). Multiplique essa chance por 10.

b) "Pelo menos 1 acerto em 5 tentativas" — de novo o truque do contrário: a única forma de NÃO ter nenhum acerto é errar as 5 vezes seguidas, chance \((0,6)^5\). A resposta é \(1\) menos isso.

I–II. Dados e classificação Distribuição binomial: \(n=5\) tentativas independentes, sucesso (acertar) com \(p=0.4\) — identificar o padrão binomial logo de início evita tentar enumerar ou somar casos manualmente.
III–IV. Resolução a) \(P(X=2) = \binom{5}{2}(0.4)^2(0.6)^3 = 10 \times 0.16 \times 0.216 = 0.3456\)
b) "Pelo menos uma vez" → complemento de "zero acertos": \[P(X\geq1) = 1-(0.6)^5 = 1-0.07776 = 0.92224\]
E se a condição fosse...Mudança no raciocínio
"No máximo uma vez" em vez de "exatamente duas"Soma \(P(X=0)+P(X=1) = (0.6)^5 + \binom{5}{1}(0.4)(0.6)^4 = 0.07776+0.2592 = 0.33696\), não apenas um único termo binomial.
7 tentativas em vez de 5 (item b)\(P(X\geq1)=1-(0.6)^7 = 1-0.0279936 = 0.9720064\) — o complemento encolhe rapidamente conforme \(n\) cresce.
Nota sobre o gabarito oficial: o PDF da Lista III lista a resposta do item (b) como 1-(0,6)⁵=0,07776. Matematicamente, \((0.6)^5 = 0.07776\), mas \(1-(0.6)^5 = 0.92224\), não \(0.07776\) — o valor no gabarito parece ser um erro de transcrição (foi anotado o valor de \((0.6)^5\), não o de \(1-(0.6)^5\), embora a fórmula escrita esteja correta). A resolução acima segue a fórmula corretamente aplicada: 0.92224. Compare com a Questão 19(b), que usa a mesma fórmula e cujo valor no gabarito (0,7599) está correto.
V. Generalização 🔁 Distribuição binomial: \(n\) tentativas independentes, sucesso com probabilidade fixa \(p\) → \(P(X=k)=\binom{n}{k}p^k(1-p)^{n-k}\); "pelo menos um sucesso" é sempre \(1-(1-p)^n\). ⚠️ Sempre calcule o complemento até o fim — \((1-p)^n\) é apenas a probabilidade de zero sucessos; "pelo menos um" exige o passo final \(1 - (1-p)^n\).
No quadro P(a)=0,4 P(e)=0,6, n=5
a) k=2 acertos: strings (a=acerto,e=erro):
aaeee, aeaee, aeeae, aeeea, eaaee, eaeae, eaeea, eeaae, eeaea, eeeaa
C(5,2)=10
P(X=2) = 10·(0,4)²·(0,6)³ = 10·0,16·0,216 = 0,3456
b) P(X≥1) = 1-(0,6)⁵ = 1-0,07776 = 0,92224

Questão 19 — Jogador de baseball, média 0.3, 4 tentativas

a) Exatamente uma vez. b) Pelo menos uma vez.
💡 Passo a passo bem simples

Mesmo tipo de questão da 18, agora com \(n=4\) tentativas e chance de acerto \(p=0,3\) (30%).

a) "Exatamente 1 acerto em 4 tentativas": sequências de 4 letras (a/e) com só um "a" — aeee, eaee, eeae, eeea — são \(\binom{4}{1}=4\) sequências. Multiplique por 0,3 (o acerto) vezes \((0,7)^3\) (os 3 erros).

b) "Pelo menos 1 acerto": \(1\) menos a chance de errar as 4 vezes seguidas, \((0,7)^4\).

Perceba que é exatamente a mesma "receita" da Questão 18 — só os números de entrada mudam (\(n\), \(k\) e \(p\)). Uma vez que você entende o padrão, os números são só substituição.

I–IV. Resolução Binomial: \(n=4\), \(p=0.3\).
a) \(P(X=1) = \binom{4}{1}(0.3)^1(0.7)^3 = 4 \times 0.3 \times 0.343 = 0.4116\)
b) \(P(X\geq1) = 1-(0.7)^4 = 1-0.2401 = 0.7599\)
E se a condição fosse...Mudança no raciocínio
"Exatamente duas vezes" em vez de "exatamente uma" (item a)\(P(X=2)=\binom{4}{2}(0.3)^2(0.7)^2 = 6\times0.09\times0.49 = 0.2646\); o coeficiente binomial muda de \(\binom41=4\) para \(\binom42=6\).
6 tentativas em vez de 4 (item b)\(P(X\geq1)=1-(0.7)^6 = 1-0.117649 = 0.882351\) — mais tentativas tornam "pelo menos um acerto" quase certo.
V. Verificação Gabarito: a) 0.4116, b) 0.7599. Bate com o gabarito.
VI. Generalização 🔁 Estrutura idêntica à Questão 18 — reconhecer o padrão binomial (\(n\) tentativas independentes, probabilidade fixa \(p\) de sucesso) é o que importa, não decorar cada exercício separadamente.
No quadro P(a)=0,3 P(e)=0,7, n=4
a) k=1 acerto: strings: aeee, eaee, eeae, eeea
C(4,1)=4
P(X=1) = 4·(0,3)¹·(0,7)³ = 4·0,3·0,343 = 0,4116
b) P(X≥1) = 1-(0,7)⁴ = 1-0,2401 = 0,7599
