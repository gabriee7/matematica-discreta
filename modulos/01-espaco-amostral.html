Módulo 1 — Espaço Amostral, Eventos e Operações de Conjuntos

Base conceitual de toda a probabilidade discreta: sem isto, nada do resto faz sentido.

Teoria

📌 O espaço amostral \(S\) é o conjunto de todos os resultados possíveis de um experimento. Um elemento particular de \(S\) é uma amostra. Um evento \(E\) é qualquer subconjunto de \(S\).

\(\emptyset\) (conjunto vazio) e \(S\) são, eles próprios, eventos: \(\emptyset\) é o evento impossível, \(S\) é o evento certo.

Como eventos são conjuntos, combinam-se com as operações usuais de conjuntos:

  • \(A \cup B\): ocorre se \(A\) ocorre, ou \(B\) ocorre, ou ambos.
  • \(A \cap B\): ocorre se \(A\) e \(B\) ocorrem simultaneamente.
  • \(A^c\) (também escrito \(\bar{A}\)): ocorre se \(A\) não ocorre.

✅ \(A\) e \(B\) são mutuamente excludentes se \(A \cap B = \emptyset\) — não podem ocorrer ao mesmo tempo.

Exemplos do professor (Exemplo 1, Aula 11)

Exemplo 1a — Lançamento de um dado

Experimento: lançamento de um dado. Seja \(A=\{2,4,6\}\), \(B=\{1,3,5\}\) e \(C=\{2,3,5\}\). Defina \(S\), \(A\cup B\), \(B\cap C\) e \(\bar{C}\).
I. Dados Um dado de 6 faces é lançado uma vez; os resultados possíveis são os números de 1 a 6 — fixar isso primeiro garante que \(S\) tenha exatamente os elementos certos antes de qualquer operação de conjuntos.
II. Classificação Definição direta de espaço amostral e operações de conjuntos — não há cálculo de probabilidade aqui, apenas a construção do modelo.
III. Resolução \(S = \{1,2,3,4,5,6\}\).
\(A \cup B = \{2,4,6\} \cup \{1,3,5\} = \{1,2,3,4,5,6\} = S\) — note que \(A\) e \(B\) particionam \(S\) (são mutuamente excludentes e cobrem todo o espaço).
\(B \cap C = \{1,3,5\} \cap \{2,3,5\} = \{3,5\}\).
\(\bar{C} = S \setminus C = \{1,4,6\}\).
IV. Generalização 🔁 \(A\) e \(B\) aqui são complementares um do outro (\(B = \bar{A}\)), por isso \(A \cup B = S\) e \(A \cap B = \emptyset\). Reconhecer quando dois eventos dados no enunciado são, na verdade, complementares evita recalcular probabilidades desnecessariamente.
No quadro \(S = \{1,2,3,4,5,6\}\)
\(A \cup B = \{1,2,3,4,5,6\} = S\)
\(B \cap C = \{3,5\}\)
\(\bar{C} = \{1,4,6\}\)

Exemplo 1b — Lançamento de uma moeda três vezes

Considerando \(C\) (cara) e \(K\) (coroa), defina \(S\). Seja \(A=\){duas ou mais caras aparecem} e \(B=\){todos os lances são iguais}. Defina \(A\), \(B\) e \(A \cap B\).
I–II. Dados e classificação Experimento composto (3 lançamentos independentes de uma moeda) — o espaço amostral é o produto cartesiano \(\{C,K\}\times\{C,K\}\times\{C,K\}\), com \(2^3=8\) resultados — usar o produto cartesiano garante contar cada sequência de resultados exatamente uma vez.
III. Resolução \(S = \{CCC, CCK, CKC, CKK, KCC, KCK, KKC, KKK\}\).
\(A = \{CCC, CCK, CKC, KCC\}\) (exatamente 2 ou 3 caras).
\(B = \{CCC, KKK\}\) (os 3 lances iguais entre si).
\(A \cap B = \{CCC\}\) — o único elemento comum é o caso de 3 caras, que está tanto em "2 ou mais caras" quanto em "todos iguais".
IV. Armadilha ❗ \(KKK\) está em \(B\) mas não em \(A\) (0 caras não é "duas ou mais"). É fácil incluir \(KKK\) em \(A\cap B\) por engano ao ler rápido demais o enunciado de \(B\).
No quadro \(S = \{CCC, CCK, CKC, CKK, KCC, KCK, KKC, KKK\}\)
\(A = \{CCC, CCK, CKC, KCC\}\)
\(B = \{CCC, KKK\}\)
\(A \cap B = \{CCC\}\)

Exercícios da Lista III

Questão 1 — Notação de conjuntos para combinações de eventos

Sejam A, B e C eventos. Reescreva usando notação de conjuntos: a) A e B, mas C não ocorre. b) A ou C, mas B não ocorre. c) Nenhum dos eventos ocorre. d) Pelo menos dois eventos ocorrem.
I. Leitura e extração de dados Não há números — o problema pede a tradução de linguagem natural para operações de conjuntos (∪, ∩, complemento).
II. Classificação 📌 Problema de tradução lógica: "e" → ∩; "ou" → ∪; "não ocorre" → complemento. "Pelo menos dois" exige decompor em casos (união de interseções par a par).
III–IV. Estratégia e resolução Traduzir: "e"→∩, "ou"→∪, "não ocorre"→complemento.
a) A ∩ B ∩ (C não ocorre) → \[A \cap B \cap C^c\] b) (A ∪ C) ∩ (B não ocorre) → \[(A \cup C) \cap B^c\] c) \(A^c \cap B^c \cap C^c\), por De Morgan = \((A\cup B\cup C)^c\) → \[(A \cup B \cup C)^c = A^c \cap B^c \cap C^c\] d) "pelo menos dois" = união dos pares → \[(A \cap B) \cup (A \cap C) \cup (B \cap C)\]
E se a condição fosse...Mudança no raciocínio
"Exatamente dois eventos"Precisa excluir o caso dos três simultâneos: \((A\cap B\cap C^c)\cup(A\cap C\cap B^c)\cup(B\cap C\cap A^c)\).
"Nenhum ou todos"União de dois casos extremos: \((A^c\cap B^c\cap C^c)\cup(A\cap B\cap C)\).
V. Verificação Confere com o gabarito oficial da lista: a) \(A\cap B\cap C^c\); b) \((A\cup C)\cap B^c\); c) \((A\cup B\cup C)^c = A^c\cap B^c\cap C^c\); d) \((A\cap B)\cup(A\cap C)\cup(B\cap C)\). Bate com o gabarito.
VI. Generalização 🔁 Este é o padrão-mestre para toda questão de "traduzir frase em conjuntos": isole cada evento nomeado, identifique o conectivo lógico (e/ou/não), e monte a expressão de dentro para fora. ❗ Armadilha comum: confundir "não A e não B" (\(A^c \cap B^c\)) com "não (A e B)" (\((A\cap B)^c = A^c \cup B^c\)) — são leis de De Morgan, e o resultado muda de ∩ para ∪.
No quadro a) \(A \cap B \cap C^c\)
b) \((A \cup C) \cap B^c\)
c) \(A^c \cap B^c \cap C^c\)
d) \((A \cap B) \cup (A \cap C) \cup (B \cap C)\)

Questão 2 — Dado e duas moedas

Um dado e duas moedas são jogados. a) Descreva um espaço amostral conveniente S e ache n(S). b) Expresse A={duas caras e um número par}, B={2 ocorre}, C={exatamente uma cara e um número ímpar}. c) Expresse (i) A e B, (ii) apenas B, (iii) B e C.
I. Dados Experimento composto: 2 moedas (K/C cada) + 1 dado (1 a 6). Resultado é uma tripla ordenada (moeda1, moeda2, dado) — descrever o resultado dessa forma evita perder informação sobre qual moeda e qual face geraram cada elemento de \(S\).
II. Classificação Espaço amostral como produto cartesiano de experimentos independentes: \[S = \{K,C\} \times \{K,C\} \times \{1,2,\dots,6\}\] — reconhecer a independência dos sub-experimentos é o que justifica montar \(S\) como produto cartesiano em vez de enumerar caso a caso.
III. Estratégia Contar \(n(S)\) pela regra do produto (\(2 \times 2 \times 6\)), depois filtrar cada evento listando os elementos que satisfazem a condição descrita — assim evita-se contar ou perder resultados ao tentar montar os eventos diretamente sem passar por \(S\).
IV. Resolução passo a passo a) \(n(S) = 2 \cdot 2 \cdot 6 = 24\)
b) resultado = (moeda1 moeda2)(número)
\(A\) = duas caras, par → \[A = \{KK2, KK4, KK6\}\] \(B\) = 2 no dado → \[B = \{KK2, KC2, CK2, CC2\}\] \(C\) = exatamente 1 cara, ímpar → \[C = \{KC1, KC3, KC5, CK1, CK3, CK5\}\] c)
(i) \(A \cap B\) → único comum = \(KK2\) → \[A \cap B = \{KK2\}\] (ii) \(B \setminus A\) → \[B \setminus A = \{KC2, CK2, CC2\}\] (iii) \(B\) exige par, \(C\) exige ímpar → sem elemento comum → \[B \cap C = \emptyset\]
E se...Mudança no caminho
Fossem 3 moedas em vez de 2\(n(S) = 2^3 \cdot 6 = 48\); listar os eventos exige mais cuidado com a combinatória de caras/coroas.
"pelo menos uma cara" em vez de "exatamente uma"Passa a incluir também KK, exigindo somar mais casos (ou usar o complemento "nenhuma cara").
V. Verificação Confere com o gabarito: \(n(S)=24\); A, B, C como acima; (i) {KK2}; (ii) {KC2,CK2,CC2}; (iii) ∅. Bate com o gabarito.
VI. Generalização 🔁 Sempre que o experimento combina sub-experimentos independentes (moedas + dado), o espaço amostral é o produto cartesiano, e \(n(S)\) é o produto das cardinalidades. ❗ \(B \cap C = \emptyset\) porque os eventos B e C impõem condições incompatíveis sobre a mesma variável (par vs. ímpar) — sempre cheque se duas condições do enunciado se referem à mesma "coordenada" do resultado antes de tentar listar interseções.
No quadro \(n(S) = 2\cdot2\cdot6 = 24\)
\(A = \{KK2, KK4, KK6\}\)
\(B = \{KK2, KC2, CK2, CC2\}\)
\(C = \{KC1, KC3, KC5, CK1, CK3, CK5\}\)
(i) \(A \cap B = \{KK2\}\)
(ii) \(B \setminus A = \{KC2, CK2, CC2\}\)
(iii) \(B \cap C = \emptyset\)
